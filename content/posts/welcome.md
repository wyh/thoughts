## 关于这个博客

这是一个使用 Hugo 构建的个人博客，主要记录我在软件工程、科学和生活方面的思考。

## 技术栈

- **静态网站生成器**: Hugo
- **主题**: PaperMod
- **容器化**: Docker
- **CI/CD**: GitHub Actions
- **托管**: GitHub Container Registry

## 开始写作

你可以使用以下命令创建新文章：

```bash
hugo new posts/your-post-title.md
```

然后编辑生成的文件，开始你的写作之旅！

## 本地预览

```bash
hugo server -D
```

访问 http://localhost:1313 即可预览你的博客。

Happy blogging! 🎉
