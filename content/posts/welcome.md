## 关于这个博客

能发表文字的地方有很多，但那天看到有人在 X 上说， 通过博客依然是最有效的记录。

我深表认同。

X 上有很多长文的限制，还需要会员。 简单一点也可以用 Medium， 但我也不是很想简单，我不是很怕麻烦。 我有时候想把简单的事情做的很复杂。

我不会考虑在中国的媒体上发表自己的观点，这主要是怕内容审查。 不仅会丢失内容，还会丢失账号。 最主要的，内容审查会封闭我的思想。

## 博客的技术栈

- **静态网站生成器**: Hugo
- **主题**: PaperMod
- **容器化**: Docker
- **CI/CD**: GitHub Actions
- **托管**: GitHub Container Registry
- **部署**: CloudFlare Tunnel

这个服务其实是部署在我家里的书房里， 这听起来不可思议，但这真是我喜欢 Cloudflare 和 github 的地方。 有时候看起来简单，但手续一点都不简单。

我使用了 hugo 来编译，并通过 github 的 actions 实现自动部署，通过 kubectl 来更新部署在我家书房的数据中心拉取最新的镜像，并通过 CloudFlare Tunnel 来保证你可以看到这个内容。

这个流程我现在已经非常熟练，但为了这个熟练实际上我的确花了很多时间。 我至少需要克服很多的翻墙的故障 （这句话在中国的社交媒体上就发布不出来）。
